# Research Data Notes

这个文件夹保存值得之后继续核实和引用的原始材料摘录。

## 已整理材料

- `key-quotes.md`：从 IEA、通风廊道论文/案例、严肃游戏案例中摘出的关键事实。

## 当前 MVP 会导出的数据字段

- `budgetUsed`：玩家使用了多少预算。
- `score`：按温度、风速、碳排计算的简化评分。
- `metrics.temp`：平均温度。
- `metrics.wind`：平均风速。
- `metrics.carbon`：碳排放。
- `layout`：每个设施的 x/y 坐标、地形、设施类型。
- `survey.budgetChoice`：是否为了节省预算选择煤电。
- `survey.windUnderstanding`：是否理解风口煤电影响通风。
