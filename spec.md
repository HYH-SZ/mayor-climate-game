# 市长模拟 — MVP Spec

## 它做什么

一个浏览器小游戏：玩家用 1000 万预算在城市网格上放置能源设施，实时看到温度、通风和碳排变化。

## 给谁用

社会科学、城市气候或 AI 仿真课堂里的大学生被试。

## 核心流程

1. 打开 `index.html`，看到 1000 万预算和 12x12 城市地图。
2. 选择煤电、风电或地源热泵。
3. 点击地图格子放置设施，观察平均温度、平均风速、碳排放变化。
4. 点击结算，查看本局评分。
5. 填 2 题反馈问卷，下载 JSON 数据。

## 不在 MVP 里

- 真实 CFD 或三维城市模型。
- 大模型 API 顾问。
- 服务器数据库、多人在线、登录系统。
- 地图编辑器、多关卡剧情。
- 热力图、通风箭头、CSV 导出、完整问卷。

## 怎么收集反馈

结算后下载一份 JSON，记录玩家布局、已用预算、三个指标、本局评分，以及 2 题问卷答案。
